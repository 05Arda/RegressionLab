Welcome to the Ultimate ML Lab. This tool is organized into three distinct sections to help you separate model parameters from data generation and validation settings.

1. MODEL HYPERPARAMETERS (Left Panel)

Controls mathematical properties of the model. These settings do not change the data.

  • Degree: Complexity of the polynomial.
    • Low (1-2) = Simple/Stiff. High (10+) = Flexible/Wiggly.
  • Alpha: Regularization strength.
    • Use this to smooth out high-degree polynomials and fight Overfitting.

2. DATA GENERATION (Center Panel)

Controls how the raw data is created.

  • Dataset Type (Radio): Choose the shape of the underlying "Truth" (Sine, Linear, etc.).
  • Noise: Adds random error to the data points.
  • Count (Size): Total number of data points.
  • New Random Seed (Button): Regenerates the random noise pattern.

3. ANALYSIS & VALIDATION (Right Panel)

Tools to evaluate how good the model is.

  • Graph Mode: Switch the right-hand plot between Complexity Curve (Degree vs Error) and Learning Curve (Size vs Error).
  • Test %: How much data is hidden from the model for testing.
  • Use Cross-Validation: Check this to use 5-Fold CV instead of a single split.
  • Residuals: Toggle red error lines on the main plot.
